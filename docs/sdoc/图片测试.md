## 1. markdown格式图片

### 1.1 不以`./`开头

```markdown
![sumu001](图片测试/img/sumu001.jpg)
```

![sumu001](图片测试/img/sumu001.jpg)

### 1.2 以`./`开头

```markdown
![sumu001](./图片测试/img/sumu001.jpg)
```

![sumu001](./图片测试/img/sumu001.jpg)

## 2. html格式图片

### 2.1 不以`./`开头

```markdown
<img src="图片测试/img/sumu001.jpg" alt="sumu001" />
```

<img src="图片测试/img/sumu001.jpg" alt="sumu001" />

### 2.2 以`./`开头

```markdown
<img src="./图片测试/img/sumu001.jpg" alt="sumu001" />
```

<img src="./图片测试/img/sumu001.jpg" alt="sumu001" />
